How to run the client and server:
1.) Open the folder as a pycharm project
2.) Right-click the server.py file and click run
3.) Right-click the client.py file and click run
What ports the server sockets are listening on:
Read Socket: Port 5000
Write Socket: Port 5001
How to send broadcast, private, and exit messages:
Private: Type "@(username) " before your message
Exit: Type "!"
Broadcast: Anything else
